Core Concepts

Essential principles and foundational concepts that underpin all machine learning systems and methodologies.

Data preprocessing is a crucial step in the machine learning pipeline. It involves cleaning and transforming raw data into a format that is suitable for a machine learning model. Key techniques include:

  • Data Cleaning: Handling missing values, correcting inconsistencies, and removing outliers.
  • Feature Scaling: Normalizing or standardizing numerical features to bring them to a similar scale.
  • Categorical Encoding: Converting categorical variables into a numerical format (e.g., one-hot encoding).

A cost function, or loss function, measures the performance of a machine learning model. It quantifies the error between the predicted values and the actual values. The goal of training a model is to minimize this function.

Activation functions are a critical component of neural networks. They introduce non-linearity into the model, allowing it to learn complex patterns. Common activation functions include:

  • Sigmoid: Maps input values to a range between 0 and 1.
  • ReLU (Rectified Linear Unit): Outputs the input directly if it is positive, otherwise, it outputs zero. It is the most widely used activation function.
  • Tanh (Hyperbolic Tangent): Similar to sigmoid, but maps inputs to a range between -1 and 1.
